1. CAPTURA CPU
================================================================================
Funcionalidad : percent global/per-core, freq, load_avg, cores
Implementación: neuromonitor/collectors/cpu.py

Datos expuestos : cores, load_avg, percent (side-channel actividad)
Controles actuales:
  + clamp_percent() NaN/Inf
  + degraded → ERR_CPU_DEGRADED (sin str(exception))

================================================================================
